Welcome to the **Threader** community!

Threader is a **visual dialogue system for Unity** — build branching conversations with a node-based graph editor, no boilerplate required.

Whether you're crafting a quick NPC conversation or a sprawling RPG dialogue tree with variables, conditions, and sub-graphs, you're in the right place.

**What Threader offers:**
🔹 Visual graph editor with an infinite canvas
🔹 15 node types — lines, choices, branching, switches, random, sub-graphs & more
🔹 Built-in variable store (Bool / Int / String) with inline conditions
🔹 Sub-graph system — call graphs from other graphs like function calls
🔹 Bark system for ambient fire-and-forget lines
🔹 Line Sheet system with multi-language support
🔹 Node templates, bookmarks, script export, edit-mode preview
🔹 Spatial audio & animator integration
🔹 Singleton `DialogueManager` — one component drives everything

## MESSAGE 7 — FAQ

# ❓ Frequently Asked Questions

**Q: What Unity versions does Threader support?**
> Check the docs for the latest compatibility info. The kit is built for modern Unity (2021+).

**Q: Do I need to write C# to use Threader?**
> Nope! The graph editor and built-in variable system let you build complex branching dialogue without writing code. C# is only needed for advanced features like custom conditions or event listeners.

**Q: Can I use Threader for commercial projects?**
> Yes — the Unity Asset Store license allows use in commercial projects.

**Q: How do I report a bug?**
> Post in `#bug-reports` with your Unity version, steps to reproduce, expected result, and actual result. You can also open a GitHub issue: https://github.com/StefanVorster17/Threader-Docs/issues

**Q: Can I translate my dialogue into multiple languages?**
> Yes! Threader has built-in multi-language support via Line Sheets. Assign one line sheet per language per graph and switch at runtime with `SetActiveLanguage()`. Full details: https://stefanvorster17.github.io/Threader-Docs/translation/

**Q: What's on the roadmap?**
> Check `#announcements` for the latest news, or see the roadmap: https://stefanvorster17.github.io/Threader-Docs/roadmap/
