    [1.1] -- Unreleased

    Added

    • Line Sheet system — per-graph companion assets (DialogueLineSheet) that store per-speaker audio clips and animator actions for every NPC line. Multiple speakers can share the same graph and each plays their own voice lines.
    • NPC nodes in the graph canvas now have a Line Data button that opens a focused per-node editor (NodeLineSheetPopup).
    • Line Sheet Editor window shows a whole-graph overview of all NPC nodes and lines; accessible from the graph editor sidebar (PROJECT → Line Sheet Editor).
    • Batch sync — Threader → Create & Sync All Line Sheets creates and syncs sheets for every graph in the project at once.
    • DialogueLineSheet cannot be created from the Project window Create menu — sheets are always created automatically alongside their graph.
    • LineSheet field on DialogueGraph is now editable in the Inspector.
    • BarkSource — Speaker Name field — a roster dropdown on BarkSource that feeds the third-level speaker resolution fallback for bark graphs. Allows a shared bark graph with no speaker set on nodes to resolve the correct voice clips and animator actions per NPC.
    • PlayBark() — speakerName parameter — optional third argument; BarkSource passes its Speaker Name automatically. Speaker resolution chain for bark graphs is now: node speaker → graph default → BarkSource speaker name.
    • DialogueGraph Inspector — redesigned with a dark header panel, stats row (nodes, NPC lines, entry points, groups), a "Used By" section showing all scene objects currently referencing the graph (clickable links that ping and select in the Hierarchy), and an editable Line Sheet field.
    • DialogueLineSheet Inspector — redesigned with stats row (lines, speakers, clips, orphaned rows), description, read-only source graph reference, and an Open Line Sheet Editor button.
    • BarkSourceEditor — custom inspector that replaces the plain Speaker Name text field with a roster dropdown, mirroring NPCDialogueEditor.
